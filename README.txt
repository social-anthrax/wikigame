Pathfinder: This will find the path between two webpages on a domain if they exist. The scraping of websites will often take a long time. \n
ReturnMap: This will show all the pages that are linked to the given page. The number of jumps given will make this recursive, and will show all the pages that are linked to the linked pages and so on. \n
DeleteTables: This will delete all the cached URLs on this system. \n

The reindex option is used to rescrape a webpage. It clears all previously scraped data off the domain.
